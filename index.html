D-Core (TG)

Dev-Core или же DC - Это персональный(Почти) проект многофункционального бота ассистента базирующегося на API: Telegram, GigaChat, YouTube, Open Weather Map.

Discord часть - предназначена для ии экспериментов и тестов новых возможнойстей Discord API, например установка анимированных аватаров и баннеров боту через api запрос.

Telegram часть - предназначена для встроенного GigaChat ai, скачивания медиа материалов с YouTube и парсинга статистики с видео.

Также, обладает мелкими фишками, такими как: счетчик подписчиков, открытие веб-приложений и интеграция с Api OWM.


Статус: в активной разработке.

D-Core

Совершенно новый проект для меня, представляет собой соц.сеть, по задумке, объединяющую как апи, так и пользовательские возможности телеграма и дискорда.

Языки: html5 (html, css, js), node.js, python.

Фреймворки: Python fastapi (Сервера) и electron js (Клиент).

Планируется создать свое открытое апи для ботов, которое будет совмещать дс и тг апи. Например разнообразные callback кнопки или модальные окна.

На данный момент занимаюсь структуриализацией сервера и клиента.
